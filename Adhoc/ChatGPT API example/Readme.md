# Readme.md

### How to build the application 

1. Epen cmd prompt in the project location 
1. Execute the following commands in cmd 
1. * ``` dotnet build ``` 
1. * ``` dotnet publish -r win-x64 ```
1. Add the ChatGPT.exe present in bin/in Environmental variables
